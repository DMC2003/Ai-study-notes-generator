# AI Study Notes Generator (GenAI Project)

This project uses Groq API to generate structured study notes from any given topic.

## 🚀 Features
- Generates:
  - Definition
  - Key Concepts
  - Real-world Applications
  - Examples
  - Summary
- Uses prompt engineering for structured output
- Simple and beginner-friendly

## 🛠️ Tech Stack
- Python
- Groq API
- Google Colab

## 📌 How it works
1. User inputs a topic
2. Prompt is sent to Groq API
3. Model generates structured notes
4. Output is displayed and optionally saved

## 💡 Example
Input: Machine Learning  
Output: Structured notes with concepts, examples, and summary
